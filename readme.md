# Requirements

  * matplotlib==3.3.0
  * numpy==1.19.1
  * SoundFile==0.10.3.post1

# Installation

```bash
# Launch a virtual environment
python3 -m venv ./venv
source ./venv/bin/activate

# Install modules
pip3 install -r requestments.txt

# run oneshot.py
python oneshot.py ./sample_data
```
